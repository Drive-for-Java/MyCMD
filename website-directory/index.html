How It Works

Get started in three simple steps

1

Download & Install

Run the setup.exe installer or build from source using Maven. Installation takes less than a minute.

2

Launch MyCMD

Open the application from your desktop or run MyCMD.exe from the command line.

3

Start Coding

Use familiar commands or extend MyCMD by adding your own custom commands in Java.

Example Usage

> dir
> cd ..
> mkdir myproject
> echo Hello World!
> type readme.txt
> help
> exit
